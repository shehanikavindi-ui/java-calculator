# java-calculator
Java-based calculator application featuring a GUI built with Swing, implementing core arithmetic operations using object-oriented programming principles.

# Features
- Basic arithmetic: + - * /
- Square root (√x), Square (x²), Reciprocal (1/x)
- Percentage, backspace, sign toggle (+/-)
- Decimal point support

# How to Run

# Requirements
- Java JDK installed ([Download here](https://www.oracle.com/java/technologies/downloads/))

# Steps
1. Download `calculator.java`
2. Open terminal/command prompt in the folder
3. Compile: `javac calculator.java`
4. Run: `java calculator`
